# Managing Plugins

## Introduction

There are currently two ways to manage plugins on NitroNodes

1. Using the built-in **plugin installer**
2. Installing them manually.

In this guide. We'll show you how to use the first way.

## Plugin installer

Go to the <a style={{ fontWeight: "bold" }} href="https://control.nitronodes.xyz"> control panel </a> and click your server to manage it.

<br />

  src="https://cdn.discordapp.com/attachments/1043294603614371920/1071398233999429742/image.png"

/>
Then on the Navbar click the **Plugins** tab.
<br />
<br />
<img

**Install**, click confirm on the modal and restart your server
