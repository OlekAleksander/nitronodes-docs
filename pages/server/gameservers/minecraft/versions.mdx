# Changing Versions

## Introduction

<Callout type="warning" emoji="⚠️">
  Changing the server jar may cause issues, or even dataloss. If you need any
  help, feel free to join our discord.
</Callout>

There are currently two ways to change server versions on NitroNodes

1. Using the built-in **version changer**
2. Installing them manually.

In this guide. We'll show you how to use the first way.

## Built-IN Version Changer

Go to the <a style={{ fontWeight: "bold" }} href="https://control.nitronodes.xyz"> control panel </a> and click your server to manage it.

<br />
<img
  src="https://cdn.discordapp.com/attachments/1043294603614371920/1071453162470375507/image.png"
  width="512"
  height="128"
/>
Then on the Navbar click the **Startup** tab.
<br />
<br />
<img
  src="https://cdn.discordapp.com/attachments/1043294603614371920/1071453257983070258/image.png"
  width="512"
  height="128"
/>
Scroll down, and look for the **MINECRAFT VERSION** field. Then input the
versions name. For example 1.19.3
<br />
<img
  src="https://cdn.discordapp.com/attachments/1043294603614371920/1071454337051668612/image.png"
  width="512"
  height="128"
/>
Then on the Navbar click the **Settings** tab
<br />
<img
  src="https://cdn.discordapp.com/attachments/1043294603614371920/1071454524419612672/image.png"
  width="512"
  height="128"
/>
Then click the **Reinstall Server** button and press **confirm** on the modal.
