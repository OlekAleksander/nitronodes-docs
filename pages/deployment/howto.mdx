# Deployment Guide

Creating a server on **NitroNodes** is really easy. In this guide we'll show you how to do it.

## Creating a server

To create a server you must go the to the storefront

Click the Shopping cart icon on the **Sidebar**

<br />
<img
  src="https://cdn.discordapp.com/attachments/1043294603614371920/1071460428976226334/image.png"
  width="256"
  height="128"
/>

Then on the **Storefront** page click the **Create** button

<br />
<img
  src="https://cdn.discordapp.com/attachments/1043294603614371920/1071460428976226334/image.png"
  width="256"
  height="128"
/>

Then on the **Storefront** page click the **Create** button

<br />
<img
  src="https://cdn.discordapp.com/attachments/1043294603614371920/1071460479047835659/image.png"
  width="256"
  height="128"
/>
Select your egg and node, then press the **Create** button. Now your server
should appear in the server list.
